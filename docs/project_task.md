# Задание проекта

## Название
ReshuLyceum

## Тема
Веб-приложение для подготовки к ЕГЭ по информатике с каталогом задач, автоматической проверкой ответов, личной статистикой и рейтингом пользователей.

## Цель
Создать учебную платформу, где ученик может зарегистрироваться, решать задачи разной сложности, получать опыт, отслеживать прогресс и видеть рекомендации задач под свой текущий уровень.

## Основные функции
- Регистрация и авторизация пользователей.
- Каталог задач с фильтрами по теме и уровню сложности.
- Страница отдельной задачи с отправкой ответа.
- Автоматическая проверка ответа.
- Начисление XP, уровней, рейтинга и достижений.
- Личный кабинет со статистикой, графиками, историей попыток и аватаром пользователя.
- Загрузка и использование файлов: пользователь может загрузить аватар профиля.
- Таблица лидеров.
- Админ-панель для создания и удаления задач.
- REST API и Swagger-документация.

## Роли
- Гость: просмотр главной страницы, каталога задач и рейтинга.
- Пользователь: решение задач, рекомендации, профиль, аватар, статистика, достижения.
- Администратор: управление задачами через админ-панель и API.

## Технологии
- Python, FastAPI, Uvicorn.
- SQLAlchemy ORM.
- SQLite.
- Jinja2 templates.
- Bootstrap, Font Awesome, CSS, JavaScript.
- Chart.js для графиков.
- JWT-авторизация.
- Docker и Docker Compose.
- pytest для тестирования.

## Критерии готовности
- Приложение запускается локально командой `python run.py`.
- База заполняется демо-данными командой `python -m scripts.seed_data`.
- API доступен на `/docs`.
- Тесты проходят командой `pytest -v`.
- Все основные страницы открываются без ошибок.
