# План презентации

## 1. Титульный слайд
Название проекта: ReshuLyceum.

Авторы: указать ФИО участников.

## 2. Идея проекта
Платформа для подготовки к ЕГЭ по информатике: задачи, автопроверка, статистика, рейтинг и рекомендации.

## 3. Для каких задач создан проект
- Помочь ученику регулярно решать задачи.
- Показывать прогресс подготовки.
- Подбирать задания подходящей сложности.
- Дать преподавателю пример полноценного веб-приложения с API и базой данных.

## 4. Основной пользовательский сценарий
- Пользователь регистрируется.
- Заходит в каталог задач.
- Фильтрует задачи по теме и сложности.
- Решает задачу.
- Получает XP, рейтинг и достижение.
- Смотрит статистику в профиле.

## 5. Реализация
- Backend: FastAPI.
- ORM: SQLAlchemy.
- База данных: SQLite.
- Frontend: Jinja2, Bootstrap, CSS, JavaScript.
- Графики: Chart.js.
- Иконки: Font Awesome.
- Авторизация: JWT.
- Тесты: pytest.
- Запуск и деплой: Docker Compose.

## 6. Демонстрация
- Главная страница.
- Регистрация и вход.
- Каталог задач и фильтр сложности.
- Решение очень лёгкой задачи.
- Профиль, графики, достижения и аватар.
- Рейтинг пользователей.
- Swagger UI на `/docs`.
- Админ-панель.

## 7. Выводы
Проект реализует основные технологии WEB-модуля: зависимости, Bootstrap, ORM, регистрацию и авторизацию, загрузку файлов, REST API, хранение данных, Docker и тесты.

## 8. Возможности развития
- Публичный хостинг.
- Импорт задач из файла.
- Избранные задачи.
- Расширенная админ-панель.
